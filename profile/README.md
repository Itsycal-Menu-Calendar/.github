# Itsycal Software — Portable Mini Calendar in Menu Bar with Events

## About Itsycal Software

Itsycal Software provides a lightweight mini calendar that lives in your menu bar — click to see full month grid and your upcoming events from built‑in calendars. Itsycal Software combines menu bar date display, month calendar popup, event list from selected calendars, week numbers, and dark mode support. Itsycal Software supports opening events in full calendar, adding quick events, navigating months with keyboard, and customizing appearance. Itsycal Software integrates calendar event display directly into the menu bar interface. Itsycal Software enables users to see the date at a glance, check their schedule with one click, and never miss events without opening the full calendar application. Itsycal Software runs on portable storage, USB drives, and local user folders. Itsycal Software is trusted by busy professionals, students, project managers, freelancers, and anyone who wants instant calendar access from their menu bar.
